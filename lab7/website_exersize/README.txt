
# Sentiment & Drug Flask App (Fixed Version)

## Steps
1. Place `sentiment_drug_app_fixed` wherever you like.
2. Run `exercise1.ipynb` and `exercise2.ipynb` (with the dump cells) **inside this folder** OR move the generated
   `artefacts_ex1/` and `artefacts_ex2/` folders so they sit next to `flask_app.py`.
3. Install dependencies:
   ```
   pip install flask scikit-learn joblib numpy pandas
   ```
4. Start server:
   ```
   python flask_app.py
   ```
5. Open <http://127.0.0.1:5000> in your browser.
